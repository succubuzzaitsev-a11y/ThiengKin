Design Principles

5 หลักการออกแบบ — ทำไม v2 ดู "โปร"

Restraint

02

Consistency

03

Real Data

04

Pro vs Amateur

อะไรทำให้ v2 "ดูโปร" กว่า v1

เปรียบเทียบ mockup v1 vs v2 — เห็นชัดว่า pro = ข้อจำกัด + ความสม่ำเสมอ

มิติ v1 (Amateur) v2 (Pro)
สีที่ใช้ 5+ สี หลายเฉด 2 สี + 4 neutrals
ตัวอักษร หลายขนาดมั่ว 8dp type scale เคร่ง
ข้อมูลใน mockup "Restaurant A" ชื่อร้านจริง · รีวิวจริง · เมนูจริง
Components แต่ละหน้าต่างกัน RestaurantCard เดียว ใช้ 10+ ครั้ง
Spacing 14, 16, 18, 20 มั่ว 8dp grid (4/8/12/16/24/32/48)
Loading state ไม่มี Skeleton + estimated time
Empty state "ไม่เจอร้าน" ออกแบบครบ (offline mode)
Phone mockup flat · no shadow real depth · island · dynamic island
Dark mode กลับด้านจาก light True dark #0a0e1a + warm amber accent
Animation hint ไม่มี pulse dot · hover lift · shimmer loading
Accessibility ไม่คิดถึง Contrast ratio · touch 56dp · focus order
