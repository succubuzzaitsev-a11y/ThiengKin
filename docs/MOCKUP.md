## 🎨 Design Principles (5 ข้อ)

| # | หลักการ | คำอธิบาย |
|---|--------|----------|
| 01 | **Restraint** | 2 สี + neutrals ไม่มี gradient ฟุ่มเฟือย |
| 02 | **Consistency** | RestaurantCard เดียว ใช้ 10+ ครั้ง ทุกหน้าจอ |
| 03 | **Real Data** | ชื่อร้าน/เมนู/รีวิวจริง ไม่ใช่ Lorem Ipsum |
| 04 | **8dp Grid** | ทุก padding/margin เป็น 4/8/12/16/24/32 ไม่มั่ว |
| 05 | **States** | Loading/Empty/Error ออกแบบครบ ไม่ใช่ afterthought |

## 🎨 Design System

### Color Palette — ธีม Thieng Tham (6 tokens + 2 semantic)

| Token | Hex | Usage |
|-------|-----|-------|
| **Primary** | `#DC2626` 🔴 | แบรนด์หลัก, ปุ่ม CTA (แดงจากโลโก้) |
| **Accent** | `#FACC15` 🟡 | Highlights, rating stars, "เด่น" badge (เหลืองจากโลโก้) |
| **Open Now** | `#16A34A` 🟢 | "เปิดอยู่" dot (semantic — เก็บไว้เข้าใจง่าย) |
| **Bg Card** | `#FFFFFF` | Light mode card |
| **Text 1** | `#14171E` | Primary text |
| **Text 2** | `#3A4256` | Secondary text |
| **Text 3** | `#6B7280` | Tertiary text |
| **Border** | `#E5E0D4` | Light borders |

**ธีม:** แดง-เหลือง จากโลโก้ Thieng Tham Development
- 🔴 **แดง** = appetite (Coca-Cola, KFC, McDonald's ใช้) — กระตุ้นความอยากกิน
- 🟡 **เหลือง** = sun, warmth — ตรงกับ "เที่ยง" (เที่ยงวัน = sun)
- 🟢 **เขียว** = เก็บไว้สำหรับ semantic "เปิดอยู่" (สากล)

**Contrast ratio** ผ่าน WCAG AA ทุกคู่

### Type Scale (8dp)

| Name | Size / Line | Usage |
|------|-------------|-------|
| display | 40/48 | Hero, branding |
| h1 | 24/32 | Page title |
| h2 | 20/28 | Card title |
| h3 | 16/24 | Sub-section |
| body | 15/24 | Body text |
| caption | 13/20 | Secondary info |

### Spacing (8dp grid)

ใช้แค่ 4/8/12/16/24/32/48/64 — ไม่มี 14, 18, 22, 28

### Dark vs Light

| Mode | When | Background |
|------|------|------------|
| **Travel · Dark** | ขับรถ | `#0a0e1a` (true dark) |
| **Near-me · Light** | จอดรถ | `#fafaf7` (off-white) |

Auto-switch ตาม mode · ไม่ใช่ toggle

## 📋 v1 (Amateur) vs v2 (Pro)

| มิติ | v1 (Amateur) | v2 (Pro) |
|------|-------------|----------|
| สีที่ใช้ | 5+ สี หลายเฉด | **2 สี + 4 neutrals** |
| ตัวอักษร | หลายขนาดมั่ว | **8dp type scale เคร่ง** |
| ข้อมูลใน mockup | "Restaurant A" | **ชื่อร้านจริง · รีวิวจริง · เมนูจริง** |
| Components | แต่ละหน้าต่างกัน | **RestaurantCard เดียว ใช้ 10+ ครั้ง** |
| Spacing | 14, 16, 18, 20 มั่ว | **8dp grid (4/8/12/16/24/32/48)** |
| Loading state | ไม่มี | **Skeleton + estimated time** |
| Empty state | "ไม่เจอร้าน" | **ออกแบบครบ (offline mode)** |
| Phone mockup | flat · no shadow | **real depth · island · dynamic island** |
| Dark mode | กลับด้านจาก light | **True dark #0a0e1a + warm amber accent** |
| Animation hint | ไม่มี | **pulse dot · hover lift · shimmer loading** |
| Accessibility | ไม่คิดถึง | **Contrast ratio · touch 56dp · focus order** |

## 🎯 Design ตัดสินใจสำคัญ

1. **ปุ่ม "🗺️ นำทาง" ใหญ่เต็มจอ** — เป้าหมายหลักคือ "นำทาง" ไม่ใช่ "ดูข้อมูล"
2. **ETA เป็น "อีก XX นาที"** — คนขับรถคิดเป็นเวลา ไม่ใช่ กม.
3. **detour โชว์ชัด** — "เสียเวลาแค่ 5 นาที" = ตัดสินใจง่าย
4. **AI Review Summary เป็น "1 บรรทัด"** — ไม่ต้องอ่าน 280 รีวิว
5. **Dark mode เฉพาะ Travel** — ขับรถกลางวัน/กลางคืน = แสงจ้า/แสงน้อย
6. **Light mode เฉพาะ Near-me** — อยู่นิ่งแล้ว อ่านง่ายกว่า

❌ AI ranking formula — ต้อง recalculate
❌ Data source — ต้อง migrate
